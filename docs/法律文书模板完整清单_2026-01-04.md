## 八、技术实现建议

### 8.1 模板结构设计

```typescript
interface DocumentTemplate {
  code: string          // 唯一编码如 "L-07"
  name: string          // 模板名称
  category: TemplateCategory
  serviceType: Array<'LITIGATION' | 'NON_LITIGATION' | 'CONSULTATION'>
  stage?: LitigationStage | NonLitigationStage
  variables: TemplateVariable[]
  content: string       // 包含 {{variable}} 占位符的模板内容
  isActive: boolean
  version: number
}

interface TemplateVariable {
  key: string
  label: string
  type: 'text' | 'date' | 'number' | 'currency' | 'select' | 'textarea'
  required: boolean
  defaultValue?: string
  options?: string[]    // select类型时的选项
}
```

### 8.2 模板内容格式

采用 Markdown + 变量占位符：

```markdown
# 民事起诉状

原告：{{plaintiff_name}}，{{plaintiff_gender}}，{{plaintiff_birth_date}}出生，
{{plaintiff_ethnicity}}族，{{plaintiff_occupation}}，
住所地：{{plaintiff_address}}，
联系电话：{{plaintiff_phone}}。

被告：{{defendant_name}}...

## 诉讼请求

{{claims}}

## 事实和理由

{{facts_and_reasons}}

此致

{{court_name}}

具状人：{{plaintiff_name}}
{{date}}
```
